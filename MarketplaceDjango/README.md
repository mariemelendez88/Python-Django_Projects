# MarketplaceDjango

Simple marketplace where people can buy and sell items, implementing:
* Authentication
* Communication between users
* Dashboard for your items
* Form handling and customizations
* and more

## Technologies
* Pycharm 2021.2.3
* Django 4.1.7
* Database: SQLite3
* HTML5
* Tailwind CSS
